IEnumerable< DialogueNode > DialogueTool.Dialogue.GetAllChildren ( DialogueNode  currentNode)

The GetAllChildren method takes one parameter to compare all children ID with the current node.

It compares all children with the current node by looking it up and comparing the ID and DialogueNode type.

Parameters
currentNodeThe currently selected node to be compared with the child nodes.
Returns
All child nodes get returned, belonging to this node (excluding the selected node).
